# Starter Fastify Template

This is a starter template for Fastify, not the best but it works.

## Features

- [x] Fastify
- [x] Typescript
- [x] MariaDB
- [x] Zod

## Getting Started

 1. Clone this repo
 2. Run `[pnpm|npm|yarn] install`
 3. Run `[pnpm|npm|yarn] run dev`
 4. Open `http://localhost:3000` in your browser
